Typography:

We have chosen "Inter" font for our website.Readability: Inter is designed with readability in mind, making it a suitable choice for both on-screen and print applications,Versatility: Inter is a versatile font that works well in various contexts, including web design, user interfaces, print materials, and more.

Multilingual Support: Inter has extensive multilingual support, covering a wide range of characters and diacritics,Open Source: Inter is an open-source font, meaning that it is freely available for use and modification.

Variable Font: Inter is available as a variable font, allowing for more flexibility in adjusting font weight and style within a single file. This can be beneficial for responsive design and dynamic content.
